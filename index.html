QR Generator

This is a simple QR code generator. Fill in the input field and the QR code will be generated. You can use this QR code to link to a website, share your email or phone number or anything else you can think of.

Features:
  • Generate QR code from text
  • Download QR code as PNG
  • Copy QR code to clipboard
